Perkenalan Tentang CSS

Maret 1, 2023 by Asari

Setelah Kita memahami apa itu HTML utnuk mempercantik halaman web kita memerlukan CSS. CSS atau juga bisa disebut ascading Style Sheets merupakan lembar format tata letak halaman web peran HTML sebagai alat struktural dan alat bantu desain terasa amat berat. Untuk itulah kemudian CSS dibuat sebagai solusi dari tugas desain dan pemformatan, sedangkan HTML bisa kembali fokus pada tata letak halaman web. CSS memiliki fungsi antara lain yaitu memuat halam situs dengan cepat, desain diproses dengan cepat dan adaptif terhadap perangkat
